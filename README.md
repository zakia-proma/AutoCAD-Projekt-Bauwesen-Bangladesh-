# Architectural Drawings with AutoCAD

## Overview
This project showcases the creation of architectural floor plans and complete construction drawings using AutoCAD.

The planning and design process was carried out independently, focusing on layout, functionality, and technical accuracy. Please note that structural calculations are not included.

## Features
- Creation of detailed floor plans
- Complete construction drawings
- Clean and precise technical drafting
- Independent project planning and design

## Tools & Technologies
- AutoCAD

## Project Scope
- Residential / building layout design
- Dimensioning and annotation
- Technical drawing standards applied

## Limitations
- No structural (static) calculations included
- Not intended for direct construction without engineering validation

## Usage
This repository is intended for:
- Portfolio presentation
- Demonstrating AutoCAD drafting skills
- Reference for architectural drawing standards
